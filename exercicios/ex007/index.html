Nesta frase temos um termo em negrito usando a tag <b> (não semântica).

Ênfase / Itálico

Nesta frase, temos um termo em ênfase usando a tag <em> (semântica).

Nesta frase, temos um termo em itálico usando a tag <i> (não semântica).

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>.

E no outro parágrafo, temos outro texto marcado no final. A cor está diferente pois foi utilizado um estilo CSS.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo (não semântica).

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado, usando a tag <del> (semântica).

Aqui estamos criando um texto excluído usando a tag <strike> (não semântica).

Texto inserido / sublinhado

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, usando a tag <ins> (semântica).

Existe também o sublinhado com a tag <u> (não semântica).

Texto sobrescrito

Para inserir coisas do tipo x20+3. Usar a tag <sup>.

Texto subscrito

Para inserir coisas do tipo H2O. Usar a tag <sub>.
